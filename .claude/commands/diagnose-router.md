---
description: Run diagnostics on a specific router by name
---

Run RouterOS diagnostics on a specific router.

Ask the user which router they want to diagnose, then execute:
`npm start -- --router "ROUTER_NAME"`

Available routers can be found in `routers.json`. Common groups include:
- WIFILINK (11 routers)
- WISP (8 routers)
- PAC (6 routers)
- SARTEK (5 routers)
- LUIS (4 routers)
- MONSTER (3 routers)
- WEBZY (2 routers)
